# lab4

Задача 1:
Реализовать класс Point для работы с точками на плоскости. Класс должен содержать координаты x и y, конструкторы, геттеры/сеттеры, метод вычисления расстояния до начала координат и перегрузку оператора вывода.

Решение:
Создан класс Point1 с приватными полями double x и double y. 
Реализованы:
  Три конструктора: по умолчанию, с параметрами, копирования
  Геттеры и сеттеры для доступа к полям
  Метод distanceToOrigin() для вычисления расстояния до начала координат
  Перегруженный оператор << для удобного вывода

Задача 2:
Расширить функциональность класса Point, добавив перегруженные операции:
Унарные: -- (уменьшение координат), - (смена координат местами)
Приведения типа: int (целая часть x), double (координата y)
Бинарные: - с целым числом, - с другой точкой (расстояние)

Решение:
Создан класс Point2 с дополнительными перегруженными операциями:
Унарные операции:
--p (префиксный) - уменьшает обе координаты на 1
p-- (постфиксный) - уменьшает обе координаты на 1
-p - возвращает точку с поменянными координатами

Операции приведения типа:
(int)p - возвращает целую часть координаты x
(double)p - возвращает координату y

Бинарные операции:
p - 5 - возвращает точку с уменьшенной координатой x
p1 - p2 - возвращает расстояние между точками
